【PConline 单机评测】NVIDIA近期发布了GeForce MX150“优选”型笔记本显卡，作为明星显卡GeForce 940MX的接班人，MX 150专为兼具工作、娱乐需求的笔记本而准备，采用备受赞誉的NVIDIA Pascal 架构，先进的工艺和构架带来了更优秀的能耗表现，配备这款显卡的笔记本既可以拥有卓越的性能，又可以具备持久的续航，非常适合入门级游戏本。
作为率先搭载GeForce MX150显卡的游戏影音本，微星PL62 7RC的表现也十分令人期待，其优雅的外观和强悍的性能更是让它有了“银色战舰”这个称号。接下来就让我们来领略一下“银色战舰”的风采。
一、舰体设计（产品外观）

　　微星PL62 7RC-005CN整机采用银色作为产品的主色调，在强光下可以散发出耀眼的金属光泽，给人一种未来科技之感，配合整机时尚简约的外观设计，很容易让人联想到《星战》中的战舰，“银色战舰”的称号可以说十分贴切。。在材质细节方面，产品的A面和C面可清楚的看到金属发丝纹质感的美型设计，就像时尚工艺品一样，用手划过可以感受到细腻顺滑的触感，从这里可以看出笔记本精湛的制作工艺。
　另一方面，触控板和键盘C面采用了一体式设计，相同的配色让二者融为一体，符合产品的整体设计风格。在触感方敏，虽然触控面板的摩擦力要大于笔记本C面，但使用过程中完全没有生涩之感，反应灵敏且定位准确，配合两个做工精致的分体式按键，实际使用体验很是出色。
blob.png
　　值得注意的是，“银色战舰”这款笔记本十分注重产品细节和用户体验方面的设计，精致的细节充分体现了微星对产品工艺精湛的把控能力，严密的边框和扎实的做工为笔记本添色不少。同时，产品在体验设计上也很贴心，较大转轴阻尼的设计可以让笔记本稳定的保持在固定角度。在产品C面，电源按钮的旁边配置了快速启动按钮（风扇配置、Dragon Center）这个按钮可以很方便的对风扇转速或系统环境进行设置，让使用者在游戏时无需为配置硬件而分心。
blob.png
　　在产品接口设置上，微星PL62 7RC将接口大部分设置在了机身左侧，右侧的接口相对较少。左侧接口分别为防盗锁孔、RJ45网口、2个USB3.0端口、HDMI端口、Mini-DisplayPort端口（Mini-DisplayPort是Display端口的缩小版，透过适当的转接器就可以使用在VGA，DVI或HDMI界面上）以及音频端口插孔（耳机和麦克风），右侧接口则是电源接口、USB2.0端口和读卡器接口，总的来说，这款笔记本的接口还是很丰富的，可以满足日常不同的扩展需求，同时，HDMI、Mini-DisplayPor等t端口的设置更是拓宽了产品的使用范围，在一些工作场景中也可以发挥作用，Matrix Display技术更是设计师和商务人士的福音。
blob.png
　　除了精致的产品外观设计外，“银色战舰”同样也继承了Prestige系列的多个优质特性，这些特性对笔记本整体能力的提升起到了很大的作用。
　　Intel Core i7 CPU 、GEFORCE MX 150、RAM（DDR4）
　　以上内容将在硬件和性能部分进行详细介绍。
　　USB3.0 Type-C
　　USB3.0提供了更高的传输规格，是一般USB2.0的20倍。USB Type-C采用双面插拔设计，比传统的Type-A更加方便，使用USB 3.0 Type-C可以更快、更方便的对游戏、电影等文件进行传输。
　　Matrix Display
　　微星是世界第一家在高阶游戏本产品上导入Matrix Display多屏幕输出功能的品牌，Prestige系列也同样支持了这项技术，透过笔记本电脑内建HDMI 1.4和Mini-DisplayPort 1.2，能够让使用者轻松地外接两台高画质屏幕输出，加上笔记本电脑的屏幕，可以提供2+1的高画质屏幕输出，使用者可以享受到多屏幕工作带来的效率提升。
　　Nahimic 2+
　　Nahimic音效强化软件能调节笔记本的声音表现，可以提供更好的音质，游戏玩家可以尽情享受具有临场感的震撼音效。（详细内容将会在下文中进行介绍）
　　Cooler Boost
　　微星P系列的笔记本电脑搭载了全新的Cooler Boost散热系统，可以提供强效的散热效率，更低的噪音值，在安静的环境中也不会被风扇产生的噪音所影响。

二、武器系统和动力系统（硬件和性能）

　　在整机硬件配置方面，这款PL62 7RC配置了Intel Core i7-7700HQ处理器，内置NVIDIA GeForce MX150独显和Intel HD Graphics 630集显，在存储设备方面，这款笔记本配备了4GB DDR4内存，1TB HDD（7200转）硬盘，从硬件配置来看，这款产品符合其游戏影音本的定位，而且搭载的最新MX150显卡也让人备受期待。
blob.png
　　本机采用了Intel Core i7-7700HQ处理器，这款处理器基于14nm的制造工艺，核心代号为Kabe Lake，拥有四核心八线程，默认主频为2.8GHz，最大睿频可达3.8GHz，6MB的三级缓存，热设计功耗（TDP）为45W。
blob.png
　　CineBench一款很有说服力的CPU和显卡测试软件，其主要是基于MAXON公司屡获殊荣的Cinema 4D特效引擎，进行跨系统和平台的性能测试。测试包括两项，分别针对显卡和处理器的性能指标。从测试结果中可看到，该处理器单核获得160cb，多核724cb，从得分排名上来看，该处理器应该是高端水平的笔记本处理器。
blob.png
　　作为“银色战舰”的一大亮点，NVDIA GeForce MX150显卡可以说是为产品增色不少，相比于Maxwell架构的GeForce 940MX，MX150在性能方面有了很大的提升，但功耗更低，更好的延长了笔记本的续航，对入门级的游戏本来说是个不错的选择。
　　在相关规格方面，GeForce MX150采用备受赞誉的NVIDIA Pascal 架构，拥有384个CUDA，16个ROP单元，64bit位宽，2GB GDDR5显存。性能方面，不仅比940MX高了30%左右，比起7代酷睿上的HD620更是达到了263%，可以说是全面碾压核显。
blob.png
　　3DMark Sky Diver 是一项DirectX 11基准测试，可用于游戏手提式电脑和中端 PC。该项目可以对主流显卡、移动设备GPU或集成显卡进行基准测试，显卡测试1侧重于曲面细分，使用向前的光照方法;显卡测试2侧重于像素处理，使用基于电脑着色器的延迟平铺光照方法。在物理测试方面，该物理模拟使用逐渐增多的雕像来测试CPU性能。
　　在Sky Diver测试中，PL62 7RC最终得到了11222的分数，其中显卡得分11970分，作为一款入门级的游戏本，总体表现还算出色。
blob.png
　　PCMark 8是一套行业标准级电脑性能测试工具，它具有专业的测试体系，可以测试从平板电脑到台式机的所有类型电脑的性能。PCMark8采用五项单独的基准测试以及电池使用寿命测试，通过实际应用任务和应用程序测评和对比电脑性能，然后将这些应用分组为不同的场景，即可反映家用电脑和工作电脑的典型使用情况，
　　在本次在测试中，我们选择PCMark 8的创作测试（Creative Test）项目来对笔记本进行测试，PCMark8 的创作基准测试（Creative Test）主要包含了媒体及娱乐内容发烧友和专业人士的常见工作负载。与家用基准测试相比（Home Test），这种测试的要求更为严格，所以适合中型电脑系统。创作测试包含网页浏览、照片编辑、视频编辑、群组视频聊天、媒体代码转码和游戏等工作负载。
blob.png
　　微星“银色战舰”在PCMark 8创作测试（Creative Test）项目下，最终得分4051。为了更好理解这个分数，我们可以结合下面的分数排名，4015这个成绩已经高于了48%的其他产品得分，而且高于一般笔记本3847的成绩，所以就整体性能而言，这款笔记本应该是一个中高端的水平。

三、实战效果（游戏测试）

　　在接下来的测试中，我们将通过一些游戏对微星PL62 7RC进行实战考验，在大型单机游戏方面，我们选择了《侠盗猎车手 5》进行测试，而在热门网络游戏方面，则选择了《守望先锋进行》作为测试项目，这两款游戏在不同的领域具有较多的受众，因此选择作为测试有一定的代表性。
　　《侠盗猎车手 5》(Grand Theft Auto V)
　　《侠盗猎车手 5》简称《GTA 5》，是一款由Rockstar Games制作并发行的动作冒险游戏。游戏采用新版雷霆引擎(RAGE引擎)，画面表现力异常强悍。游戏故事发生在以美国洛杉矶、南加州及其周边地区为原型的城市洛圣都。制作单位拍摄了超过25万张相关照片，并且研究了人口调查和汽车销售数据，以建立游戏中的世界。
blob.png
　　GTA 5是一款对性能要求非常严苛的游戏，我们将游戏画质设置到了中高等级别，微星PL62 7RC可以流畅运行。
blob.png
　　《守望先锋》
　　《守望先锋》是一款由暴雪公司制作并发行的第一人射击游戏。虽然是FPS游戏，但并没有一味的追求画面的质量，暴雪更在意培养玩家的团队合作精神。游戏的战斗节奏很快，而人物的定位也大致分为四种，分别为攻击，防守，肉盾以及辅助，玩家必须通力合作才可赢得比赛的胜利，个人主义在这里行不通。
blob.png
　　在游戏配置方面，在1920X1080的分辨率下，我们将图象质量设置到极高，特效开到高等水平。
blob.png
　　从实际游戏效果来看，在极高画质下，游戏运行十分顺畅，平均帧数在60帧左右，运行主流网络游戏看来没有太大的压力。

四、排水量（散热能力）

　　微星PL62 7RC在底部和机身后部设有多处散热孔，尤其是在机身底部的大面积通风孔设计，对整机的散热帮助很大。同时，配合Cooler Boost散热系统，让人对产品的散热能力有了很高的期待。
blob.png
　　AIDA64是一款测试软硬件系统信息的工具，它不仅提供了诸如协助超频，硬件侦错，压力测试和传感器监测等多种功能，而且还可以对处理器，系统内存和磁盘驱动器的性能进行全面评估。我们将选用AIDA64的系统稳定性测试（System Stability Test）对笔记本散热进行测试，在高负载的情况下，AIDA64可以根据实时更新的图表测试温度，风扇转速，电压来监测散热性能，在进行强度测试时，测试数据能够用于注册传感器显示的最低，最高和平均值。
　　在将近一个小时左右的高压负载下，“银色战舰”的散热依然保持良好，CPU温度维持在平均85度左右。
blob.png
　　另一方面，根据测温仪生成的图象可以看到，在室温29度的环境下，本机在高负载的情况下，机身温度最高达到了46.6度，温度还算适中。虽然主要高温集中在键盘附近，但在实际使用过程中，并没有过热的感觉。
blob.png
　　由于D面设置了很多通风孔，所以在图上可以看到围绕通风孔的多个位置出现了红色的痕迹，这间接说明了本机良好的散热能力，而最高44.2的温度也是可以接受的。

五、特色功能和评测总结

1、特色功能
　　Dragon Center是一款功能强大的工具型软件，可以对系统环境进行监测和设置，以及对移动端APP的远程控制。用户不需要打开多个软件，就可对微星自带的各个工具进行整体调度，使用起来非常方便。
blob.png
　　Nahimic音效软件一直是微星游戏本的特色功能之一，其支持360°虚拟环绕声输出体验，同时也提供了更清晰的录音功能。Nahimic 2是Nahimic音效软件的升级版，除了支持Nahimic 原有的音频优化功能之外，还提供了专为游戏玩家设计的“声音追踪器”功能，让玩家在游戏中通过视觉更直观的判定出对手的位置。
blob.png
　　Shortcut Manager这款软件可以根据用户自己的需要，对热键进行设置，有助于提高工作效率，并且获得更加便利的使用体验。
blob.png
2、总结
　　不得不说，微星PL62 7RC(银色战舰）的外观设计十分出色，细腻的材质充满了时尚工艺品的质感，极具科技感的造型更是令人印象深刻。在硬件配置上，i7-7700HQ和MX150的组合可以满足大多数游戏的配置要求，良好的散热能也保障了运行的流畅性。同时，这款游戏本也展现出了强大的办公能力，Matrix Display 技术、Mini-DisplayPor端口的设置更是极大的提高了工作效率，Dragon Center、Nahimic等特色工具也可以很方便对笔记本进行管理和设置。总的来说，微星“银色战舰”是一款非常全面的游戏本。
